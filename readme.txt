1.Compilation:
Compile the code using a C++ compiler (e.g., g++, DevC++).

2.Execution:
Run the compiled executable.

3.Gameplay:
Follow the on-screen prompts to input row and column numbers for your moves.
The game will display the updated board after each move.

4.End of Game:
The game will declare a winner if a player achieves a winning condition.
If no winner is found after 25 moves, the game declares a draw.
